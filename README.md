# embedchain-chat-app-frontend
Embedchain-chatapp-frontend

## Install dependencies
```
npm install
```

## Change server port 
```
// utils/axiosConfig.js
baseURL: 'http://localhost:5000'
```

## Run server
```
npm run dev
```

## CORS issues
